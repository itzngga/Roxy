# Roxy
a Bot builded with Typescript and Multidatabase support

## MultiDB
##### currently supported databases
- json
- mysql
- postgress
- sqlite3

## Instalation
- `npm i` to install development modules
or
- `npm i --production`  to install production modules

- `npm run build` to build typescript

## Usage
- `npm run start` standart mode
- `npm run lint` eslint
- `npm run dev` ts-node-dev mode
- `npm run dev` pm2 mode

## Config
copy from existing [env](https://github.com/ItzNgga/Roxy/blob/master/.env.example) file

## Requirements
- Android/Termux currently not supported
- ImageMagic/GraphicsMagic support
- FFMPEG support
